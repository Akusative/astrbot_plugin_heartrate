# 💓 AstrBot 心率监听全指南：多平台硬件接入教程

无论你使用的是 Apple Watch 还是小米手环，本教程都将引导你完成从硬件设置到机器人接收心跳的完整流程。让你的 AI 伴侣实时感知你的每一次“心动”！

---

## 🛠️ 第一步：通用准备工作

无论使用哪种设备，首先请确保 AstrBot 侧的服务已就绪：

1. **更新 AstrBot 插件**：确保你已在 AstrBot 安装了本插件 `astrbot_plugin_heartrate`。
2. **确认服务运行**：在聊天对话框中发送 `/监控状态` 或 `/重启心率服务`，确认监听服务（默认端口 `3476`）正在运行。
3. **确认网络**：你的手机或手表必须能够访问 AstrBot 所在电脑/服务器的公网 IP 及对应端口。请确保云服务器的防火墙或本地电脑的入站规则已放行 `3476` 端口。

---

## 🍎 方案一：Apple Watch 接入指南

针对 Apple Watch 用户，我们推荐使用 App Store 中的 **Health Data Server (HDS)** 软件进行后台推送。

### 1. 下载与配置 HDS App
1. 在 iPhone 的 App Store 中搜索并下载 **Health Data Server**。
2. 打开 App，授权其读取「健康」App 中的**心率**数据。
3. 进入 HDS 的设置 (Settings) 页面，找到 **Server URL** 或 **Webhook** 配置项。
4. 填入你的 AstrBot 心跳服务地址：`http://你的公网IP:3476`。
5. 设置请求方式 (Method) 为 **PUT** 或 **POST**。

### 2. 在 Apple Watch 上开启同步
1. 在 Apple Watch 上打开 Health Data Server 应用。
2. 点击 **Start** 按钮开始实时监测与推送。
3. *注：为保证后台持续推送，建议在 Apple Watch 的设置中开启该 App 的后台刷新权限，或在锻炼时开启体能训练模式。*

## 🍊 方案二：支持心率广播的智能穿戴设备（小米/华为/华米等）接入指南

从 **CatlabPing v3.4.1** 开始，我们支持了 **原生 GATT 协议解析 (BLE 直连)**，不再需要安装 Notify for Mi Band 等第三方破解软件！不限制于小米手环，只要你的设备（包含具备心率广播功能的华为手表等）支持标准的心率服务 (UUID `180D`)，就能用 Android 手机直接连接并获取 BPM。

### 1. 开启设备的「心率广播」
1. 打开你平时用来同步手环/手表的官方 App（如 **小米运动健康**、**Zepp Life**、**华为运动健康** 等）。
2. 进入设备的 **设备设置** → **心率监测 / 健康监测**。
3. 确保心率监测模式已开启（推荐选为“实时/智能监测”）。
4. 向下滑动，找到 **【心率广播】(Heart Rate Broadcast)** 功能，将其开启。
   - *（可选记录）* 记住设备的蓝牙 MAC 地址（如 `EB:12:34:56:78:90`），配置时填入连接会更迅速。

### 2. 配置 CatlabPing 直连参数
1. 前往 CatlabPing 仓库下载安装 **v3.4.1+** 版本的 Android App。
   - 👉 [点击直接下载 CatlabPing_v3.4.1_debug.apk](https://github.com/Akusative/CatlabPing/releases/download/v3.4.1/CatlabPing_v3.4.1_debug.apk)
2. 打开 App，在首页找到 **💖 心率自动监听** 卡片，点击 **⚙️ 设置**。
3. **填写服务器地址**：`http://你的公网IP:3476`。
4. **勾选【直接连接 BLE 设备】** 开启原生蓝牙直连模式。
5. **填写设备 MAC 地址**（可选但推荐）：填入你刚才记录的 MAC 地址，留空则自动扫描附近开启广播的设备。
6. 点击 **💾 保存设置**。

### 3. 授权与防杀后台（非常重要）
1. 在 CatlabPing 首页打开 **💖 心率自动监听** 的主开关。
2. **首次授权**：系统会提示需要「位置信息」和「附近的设备（蓝牙）」权限，这是安卓系统扫描蓝牙的强制要求，请务必选择 **始终允许**。
3. **解除电池优化**（以小米澎湃 OS 为例）：
   - 进入 **设置** → **应用设置** → **应用管理** → 找到 **CatlabPing**。
   - **省电策略** 改为 **「无限制」**。
   - 开启 **「自启动」**，并在多任务列表里给 CatlabPing 加上 **🔒 锁头**。

## 🎉 享受心跳联动！

一切准备就绪！现在，你可以在与 AstrBot 的聊天窗口中体验以下交互：

1. 发送 **“我今天心率怎么样？”** 或者 **“帮我看一下我紧张吗”**。
2. 机器人会悄悄调用 `get_heartrate` 工具，从服务端的 3476 端口拉取到你的实时 BPM 数据。
3. 它会根据你当前的心率状态（如静息、中度兴奋、剧烈运动等），用它专属的人设语气给你回复！

> **💡 调试小贴士**：
> 如果你感觉连不上，可以随时在电脑端浏览器访问 `http://你的公网IP:3476/status`。页面会以醒目的大字展示当前是否在线，以及最新一条收到的 BPM 数值。只要这里有数据跳动，你的机器人就能感受到你的心跳！
